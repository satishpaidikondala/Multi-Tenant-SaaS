# Product Requirements Document (PRD)

## User Personas

1. **Super Admin:** Manages the entire platform and tenant billing.
2. **Tenant Admin:** Manages organization users and projects.
3. **User:** Manages assigned tasks within their organization.

## Functional Requirements

- FR-001: System shall allow tenant registration with a unique subdomain.
- FR-002: System shall enforce max project limits per subscription plan.
- FR-003: Users shall only see data belonging to their own `tenant_id`.
